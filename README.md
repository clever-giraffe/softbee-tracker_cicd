# softbee-tracker

Custom time tracker with next functions:
- live time tracker
- report by date, projects, clients, users
- vacation approval
- user profile
- add/edit/delete user
- crew page
- add/edit/delete project

The main tech stack:
## Frontend
- React
- Material UI
- TypeScript
- JavaScript

## BackEnd
- Node.js
- Strapi

## API
- GraphQL + Apollo

## DB
- PostgreSQL
